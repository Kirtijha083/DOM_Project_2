# 📋 DOM Practice Project

# 🚀 Overview

This project showcases DOM manipulation techniques using HTML, CSS, and JavaScript. It features an interactive Instagram-like double-tap heart animation for photo interactions.


# 🎯 Features

1. **Heart Animation on Double Click:** A heart animation appears when the photo is double-clicked, mimicking Instagram's like feature.

2. **Event Listener Implementation:** Uses JavaScript event listeners to detect user interactions, ensuring smooth functionality.

3. **Timeout Functionality:** Implements *setTimeout()* to control the animation's timing, enhancing the visual experience.


# 🛠️ Technologies Used

1. *HTML*

2. *CSS*

3. *JavaScript (DOM Manipulation)*
